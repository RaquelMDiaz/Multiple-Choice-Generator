Multiple Choice Test Generator

This application will allow you to create, edit and delete multiple choice tests and store them in the related data base.

Follow the instructions as to how to complete the different input fields. Remember that it is not possible to store tests that do not contain a name or at least one question. Likewise, it is not possible to input the same question twice in the same test.
